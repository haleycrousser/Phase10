HOW TO PLAY: (2-6 players)

The game ends when a player reaches phase 10, and the winner is determined by the player with the lowest points.

Each phase requires a specific card combination. A player must complete the current phase before moving on to the next.

Players must complete these phases in order:

2 Sets of 3

1 Set of 3 + 1 Run of 4

1 Set of 4 + 1 Run of 4

1 Run of 7

1 Run of 8

1 Run of 9

2 Sets of 4

7 Cards of One Color

1 Set of 5 + 1 Set of 2

1 Set of 5 + 1 Set of 3



Round Setup:

Shuffle the deck.

Deal 10 cards to each player.

Place the remaining cards face down (Draw Pile).

Flip one card face up to start the Discard Pile.



Turn Structure:

Draw one card from the Draw Pile or Discard Pile

Attempt to complete the current phase (if possible).

If the phase is completed, lay it down on the table.

Discard one card to end the turn.

If you have layed the cards down, you are able to add onto other players placed down cards.

Winning of round is the first person to 0 points.



Wild Cards

Can represent any number or color.

Once played, they cannot be changed.

Worth 25 points if left in hand at the end of a round.


Skip Cards

Causes a chosen player to lose their next turn.

Worth 15 points if left in hand.



End of round:

Points are added up with these constraints:

| Card Type | Points |
| --------- | ------ |
| 1–9       | 5      |
| 10–12     | 10     |
| Skip      | 15     |
| Wild      | 25     |
